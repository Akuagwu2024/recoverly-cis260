Recoverly

A simple Lost & Found hub for the City Colleges of Chicago. Built by students to help students, staff and faculty recover misplaced items.

This prototype stores data in the browser so it is safe to demo. A campus deployment can add a shared backend if desired.

Purpose

Report

Post a concise report for a lost or found item with contact details so matches can be made.

Search

Search saved reports by keyword, location, or status (lost / found).

Contact

Leave site feedback or notify the project lead about recovered items.
